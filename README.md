1️⃣ What is the difference between var, let, and const?
ans : In JavaScript, var, let, and const are all used to declare variables. But they have differences in scope, hoisting, and reassignment.
i.var is the old way.
ii.let came in ES6.
ii.const is for constant variables.

2️⃣ What is the spread operator (...)?
ans : Spread Operator (...) is used to expand elements of an array, object, or iterable into individual elements.

3️⃣ What is the difference between map(), filter(), and forEach()?
ans : map() operates on each element of an array and returns a new array.
filter() selects some elements according to a condition and returns a new array.
forEach() simply loops over each element of the array.

4️⃣ What is an arrow function?
ans : Arrow Function (=>) is a shorter and cleaner way to declare a function.

5️⃣ What are template literals?
ans : Template Literals are a modern (ES6) way of creating strings, where we can easily write variables, expressions, and multi-line strings.
